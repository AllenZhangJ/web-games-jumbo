# Arena Presentation Runtime

Arena 的宿主无关表现运行时原语。该包拥有有界事件窗口、质量档位、渲染节拍、固定 tick 累加器、单帧调度、六方向解析、输入语义映射和资产加载 lease 生命周期，但不拥有 Match、Renderer、Three.js、DOM 或平台资源。

宿主调度、墙钟、错误观察器和资产加载必须通过显式端口注入；所有输入在状态变化前验证，迟到帧通过 generation 失效，加载完成后的资源只能通过同步 lease 释放。原始触控、手势识别、语义映射与固定 tick 采样使用同一包内所有权链；正常路径通过私有可信快照避免重复深校验，外部或伪造对象仍按完整边界复验。三套生产输入映射的攻击意图不读取敌人距离，命中资格仍只由权威 Rule/Core 裁决。
