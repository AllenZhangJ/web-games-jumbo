## Arena Product 抖音预验收补充（2026-07-20）

抖音开发者工具 V4.5.3 已在 iPhone 15 Pro / iOS 15 模拟器中导入并运行当前 `dist/douyin` Product dirty 构建。393×852、DPR 3、安全区环境下，主页、正式 GLTF 角色、手持武器/盾牌、屏外地图、移动摇杆、攻击/跳跃、强制刷新和重新入局均正常；Console 未见项目错误。该结果仅为开发者工具预检，不勾选下方旧 v3 真机项目，也不替代 clean-build Record。完整记录见 [Arena Stage 6 抖音开发者工具预验收记录](research/arena-stage6-douyin-developer-tool-preflight.md)。

项目方随后使用 iPhone 13 Pro / iOS 26 / Chrome 对局域网 Web Product dirty 构建人工试玩：无黑屏，确认打击感、胜负闭环和低上手门槛；报告约 `1/10` 概率卡顿、设备明显发热、动作/二段跳不够灵活、基础攻击距离门控、少量穿插以及操作上限不足，综合评分 `4.3/10`。问题与修复追踪见 [Stage 7 正式资产结果](research/arena-stage7-formal-asset-intake-results.md#2026-07-20-手机人工预验收基线)。

## 共用玩法与动作

- [x] 本机浏览器验证只有底部 `↖` / `↗` 按钮可分别锁定屏幕左/右候选；按钮高亮和实际飞行方向一致，点击场景不蓄力，一次操作只跟踪一个触点。
- [ ] 按住期间角色变矮变宽，当前平台同步受压；反馈连续且与按住时长一致。
- [ ] 松开后角色和平台回弹，角色沿 Core 抛物线起跳并完成局部前空翻。
- [ ] 角色根位置、地面阴影和拖尾方向与实际轨迹一致，没有视觉吸附。
- [ ] 成功落地只校正高度，角色在平台上的真实偏心 `x/z` 位置保留。
- [ ] 相同 seed 与相同蓄力时长在不同刷新率下获得相同碰撞结果。
- [ ] 成功落地后所选平台保持 ID 并晋升，未选分支退场，新左右候选生成。
- [x] 本机浏览器采样确认世界原点与镜头焦点经过 `80 ms` 停留和 `440 ms` 同步位移平滑前进；视觉平移不会改变下一跳距离。
- [ ] 成功落地后才执行运算与扣步数；短跳、越过和边缘失败不修改数值或步数。
- [ ] 短跳、越过和边缘失败有区分度，倾倒/滑落/坠落动画不把失败修正为成功。
- [ ] 暂停冻结输入提交、Core 更新和表现时间轴；恢复后不追赶大段后台时间。
- [ ] 重开、胜利下一局和失败重开都不重复绑定输入、帧循环或上下文事件。

- [ ] 上下文丢失测试能停止错误渲染；恢复后 Scene 从 Core 快照重建，数值与世界不倒退。

## 性能和资源

- [ ] 记录待机、蓄力、空中、粒子爆发和连续 10 分钟后的 FPS、draw calls、三角形和内存。
- [x] 本机 Chrome 手机视口短样本显示逐帧缓存复用后 JS 堆增量由优化前约 `4.30MB/2.03s` 降至约 `0.39MB/5.03s`；样本不替代真机热量和长稳测试。
- [ ] iPhone 13 Pro 贴身连续攻击 30 次时，命中特效使用预热对象池且程序化角色插值无逐帧临时 Vector3；性能记录能区分设计 hit-stop 与非预期 `>50 ms` 长帧。
- [ ] 目标中端设备常态 55–60 FPS；如果未达到，附采样证据和降级决策。
- [ ] 常态 draw calls 目标不超过 50，三角形目标不超过 30k，纹理显存目标低于 32MB；以采样而不是代码推断勾选。
- [ ] 多次重开、多局连续进行和切后台后，Mesh、Material、Texture、粒子和拖尾数量回到有界稳态。
- [ ] 阴影降级在低端设备上生效且不改变碰撞或 HUD 可读性。

## 异常处理

- [ ] WebGL2 不可用：显示平台标识和可操作建议，保留诊断日志。
- [ ] 资源加载失败：不进入可交互状态，不留下无反馈黑屏。
- [ ] 触摸取消/来电/强制切后台：清理本次蓄力，不意外提交跳跃。
- [ ] 构建、导入或真机阻断时：保留原始错误和复现步骤，不将未验证项勾选为通过。
