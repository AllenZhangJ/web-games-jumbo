## 性能基线

在 Node v20.19.5 / Apple Silicon 上连续 5 次构造 10,000 个 `normal@1` 初始回合，样本为 1212.82、1226.24、1162.62、1039.88、1058.88 ms，中位数 1162.62 ms。

该数据只用于发现数量级退化，不作为跨机器硬门禁。浏览器帧率、内存和温度仍需要真实设备测量，当前没有把本机 Node 吞吐误写成移动端性能结论。

## 使用规则

1. 修复 bug 若必须改变基线，先说明旧行为为何错误并新增 ADR 或变更记录。
2. 基线更新必须与实现和测试同一提交，不能只改 fixture 让测试变绿。
3. 每个正式版本新增独立 fixture，不覆盖旧版本，以便未来回放和存档迁移。
