# 变更日志

本项目遵循“每个治理批次独立提交、标签与验收证据”的维护方式。版本发布前先更新 `Unreleased`，发布时再移动到对应版本。

## Unreleased

### 新增

- 可扩展的玩法、任务、难度、场景、角色、反馈、平台与存档注册边界。
- 版本化本地存档、前两版迁移、确定性回放和本地诊断导出。
- 1000 会话与 100 局资源 soak、覆盖率、资产/许可证、零旧 JS 和三端构建门禁。
- GitHub Actions、CODEOWNERS 与发布清单。

### 变更

- 维护源码、测试和构建工具全部迁移到 strict TypeScript。
- 游戏真相由核心/应用层拥有，Three.js 仅消费快照和事件。

### 兼容性

- P0–P8 冻结的数值策略、世界状态、碰撞规则与左右按钮行为保持不变。
- Web、微信小游戏和抖音小游戏仍输出独立产物。
